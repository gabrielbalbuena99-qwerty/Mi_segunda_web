# Mi_segunda_web
Web de la unidad numero 6
Durante la Unidad 6 trabajamos en el desarrollo de una página web aplicando los contenidos vistos en clase. Analizamos distintos proyectos que el profesor nos mostró mediante el proyector, observando cómo cada uno implementaba herramientas de inteligencia artificial. A partir de esos ejemplos, fuimos adaptando las ideas y técnicas a nuestros propios proyectos, hasta llegar al resultado final: una página funcional y personalizada que refleja lo aprendido durante la cursada
